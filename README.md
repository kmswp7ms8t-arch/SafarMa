# SafarMa | سفرِ ما

A bilingual worldwide couple-trip planner created by Amir as a birthday gift for Sanaz.

## Live app

**https://kmswp7ms8t-arch.github.io/SafarMa/**

## Install on iPhone

Open the live link in Safari, tap **Share**, choose **Add to Home Screen**, enable **Open as Web App**, and tap **Add**.

## نصب روی آیفون

لینک بالا را در Safari باز کنید، Share را بزنید و **Add to Home Screen** را انتخاب کنید.

The public GitHub Pages edition performs in-browser destination, entry-rule, safety, availability, tourism and video research. It shows sources and avoids inventing unresolved rules. Secure commercial provider credentials are intentionally not stored in this public repository.
